# Login Form UI

## Description
This project is a simple **login form user interface** created as a frontend practice exercise.
It focuses on form design, layout, and styling using basic web technologies.
The form does not store, validate, or process user data.

## Features
- Username and password input fields
- Clean and simple UI design
- Frontend-only implementation

## Technologies Used
- HTML
- CSS

## Project Purpose
- Practice creating forms using HTML
- Improve CSS styling and layout skills
- Understand basic structure of authentication UI

## Limitations
- No backend integration
- No data storage or authentication
- No form validation logic

## Future Enhancements
- Add form validation using JavaScript
- Connect to a backend for authentication
- Improve UI/UX with animations
